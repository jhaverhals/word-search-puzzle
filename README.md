# Word Search Puzzle

Word search (woordzoeker in NL) puzzle check functionality

## Local setup

```bash
# Initialize and install dependencies
npm i

# Run continuous builds
npm run build:watch

# Run continuous tests (in parallel terminal)
npm run test:watch

# Start web server (in parallel terminal)
npm run serve
```
